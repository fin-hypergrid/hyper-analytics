Class: DataSourceDepthSorter

DataSourceDepthSorter

Sorts on non-terminal tree node rows only (i.e., expandable rows with children).

One of these sorters is created by DataSourceTreeviewSorter for each grouping level, starting with the maximum group level depth, and then one for each group level through the top level (0) sort depth.

sortOn(groupLevel, directionopt, columnIndexopt)

Stable-sorts non-terminal tree node rows. Terminal (leaf) rows remain stable.

Parameters:
Name Type Attributes Default Description
groupLevel number

If greater than row depth, sorts on an edge value value, which is a value lexically inferior to (ascending sort) or superior to (descending sort) the row value. Otherwise sorts on value of ancestor of this depth.
direction number <optional>
 1

One of: 1 - Sort ascending. -1 - Sort descending.
columnIndex number <optional>

Sorts on the values in this column. Otherwise sorts on the row index.
